Card Checker with IP address Tracker

This application is only for Demo purpose and do not store any data.




THe Web application uses the following web services. You can get more details about the services by clicking on the URL.
Service Name Description URL
IP2Geo This service can accept the ip address and give the detailed information about the location such as country, city, latitude, etc. For the testing purpose we need to set the licence key to 0. Link to WSDL
CreditCardValidator This service can accept the card number and give information such as card type, validity, valid length period, etc. There is a validCard method in the service, however this needs two parameters including expiration date for the execution. Therefore, I created the method to validate the card length in the servlet. Link to WSDL
